Description

Provides tools for calculating and managing cargo crates divided by size.

Currently, the sizes available in SCU are: 32, 24, 16, 8, 4, 2, 1.

(SCU - Standard Cargo Unit)

1 SCU ≡ 1.25m x 1.25m x 1.25m = 1.953125m3
